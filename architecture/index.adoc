== Glossary

**SPA (Single Page Application):**
A web application that loads a single HTML page and dynamically updates content as the user interacts with the app.

**JWT (JSON Web Token):**
A compact, URL-safe means of representing claims to be transferred between two parties. Used for authentication.

**Hexagonal Architecture:**
An architectural pattern that separates business logic from external concerns through ports (interfaces) and adapters (implementations).

**GDPR (General Data Protection Regulation):**
EU regulation on data protection and privacy in the European Union and the European Economic Area.

**K3s:**
A lightweight Kubernetes distribution designed for resource-constrained environments and edge computing.

**Drizzle ORM:**
A TypeScript ORM with a focus on type safety and developer experience.

**Bun:**
A fast all-in-one JavaScript runtime and toolkit designed as a drop-in replacement for Node.js.
